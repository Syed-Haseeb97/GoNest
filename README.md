# GoNest

A subscription-based recurring commute platform built to simplify daily transportation.

## 🚀 Vision

GoNest is designed to help students, office employees, teachers, and everyday commuters create recurring rides instead of booking transportation every single day.

Instead of opening a ride-hailing app every morning, users subscribe to their daily commute while drivers receive predictable recurring passengers.

The goal is to make commuting simpler, more reliable, and more affordable.

---

## 💡 The Problem

Millions of people book the same trip every day.

- School → Home
- Home → Office
- Office → Home

Traditional ride-hailing apps treat every ride as a brand-new booking.

GoNest changes that.

---

## ✨ The Solution

Users can:

- Create recurring daily rides
- Subscribe to fixed commute schedules
- Manage recurring bookings easily

Drivers benefit from:

- Predictable income
- Regular passengers
- Better route planning
